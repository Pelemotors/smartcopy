# תמונות לטכנאי שיניים

תיקייה זו מכילה תמונות רלוונטיות לפרויקט טכנאי השיניים.

## תמונות נדרשות

יש להוריד ולהוסיף את התמונות הבאות לתיקייה זו:

### תמונות ראשיות:
1. **dental-lab-hero.jpg** - תמונה ראשית לעמוד הבית (מומלץ: 1200x800px)
   - תמונה של מעבדת שיניים או טכנאי שיניים בעבודה
   - מקורות: Unsplash "dental lab", Pexels "dental laboratory"

2. **dental-work.jpg** - תמונה של עבודת שיניים (מומלץ: 800x600px)
   - תמונה של תותבות, כתרים או עבודות שיניים
   - מקורות: Unsplash "dental work", Pexels "dental crown"

3. **dental-process.jpg** - תמונה של תהליך העבודה (מומלץ: 800x600px)
   - תמונה של תהליך יצירת תותבות/כתרים, ציוד CAD/CAM
   - מקורות: Unsplash "dental process", Pexels "dental technician"

4. **dental-lab-about.jpg** - תמונה לעמוד אודות (מומלץ: 1200x600px)
   - תמונה של המעבדה או הצוות
   - מקורות: Unsplash "dental laboratory", Pexels "dental lab"

5. **dental-technician.jpg** - תמונה של טכנאי שיניים (מומלץ: 600x600px)
   - תמונה מקצועית של הטכנאי בעבודה
   - מקורות: Unsplash "dental technician", Pexels "dental technician"

6. **dental-smile.jpg** - תמונה של חיוך בריא (מומלץ: 400x400px)
   - תמונה של חיוך יפה ובריא
   - מקורות: Unsplash "dental smile", Pexels "perfect smile"

## הוראות הורדה

1. גש לאחד מהמקורות לעיל
2. חפש תמונות רלוונטיות
3. הורד תמונות באיכות גבוהה (לפחות 1200px רוחב לתמונות hero)
4. שמור את התמונות בשמות המדויקים שצוינו לעיל
5. שמור תמונות before/after בתיקיית `before-after/`

## הערות חשובות

- כל התמונות צריכות להיות בפורמט JPG או PNG
- מומלץ לאופטימיזציה של תמונות לפני העלאה (הקטנת גודל קובץ)
- יש לוודא שיש רישיון שימוש חופשי לתמונות (Unsplash, Pexels, Pixabay - כולם חינמיים)
- התמונות צריכות להיות איכותיות ומקצועיות
- חשוב שהתמונות יהיו רלוונטיות לנושא - מעבדת שיניים, טכנאי שיניים, ציוד דנטלי

## עדכון קבצים
